Installation ğŸ”§

To install the project locally and run the api please execute the following steps

Clone the api-node-characters repository from github

git clone https://github.com/desobsesor/api-node-characters.git

Install the dependencies

npm i

Boot Ubuntu on Windows and its redis service

sudo service redis-server start

Consume the post method in the character/init path from Swagger or execute the following CURL 

curl --location --request POST 'http://localhost:4000/character/init'

Compile the project and start the server

npm run start-dev

Invoke the service status endpoint

http://localhost:4000
